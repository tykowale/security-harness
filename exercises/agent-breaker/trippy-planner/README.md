# Trippy Planner

Notes from the Gandalf Agent Breaker Trippy Planner challenges.

## Levels

- [Level 1](level-1.md) - poisoned travel webpage influences a generated Armenia itinerary.

## Recurring lessons

- Web search results and travel blogs are untrusted data, not planner instructions.
- Copy-ready blocks, required notices, and exact-preservation language are high-risk in retrieved content.
- Booking/payment links should be validated or excluded from generated itineraries unless they come from trusted sources.
