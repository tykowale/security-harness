# Curs-ed CodeReview

Notes from the Gandalf Agent Breaker Curs-ed CodeReview challenges.

## Levels

- [Level 1](level-1.md) - poisoned repository guidance steers a code review recommendation.
- [Level 2](level-2.md) - literal-preservation and audit-tooling framing steer a more dangerous recommendation.

## Recurring lessons

- Review assistants must distinguish trusted review policy from untrusted repository content.
- Plausible compliance language can make poisoned rules look legitimate.
- Literal-preservation instructions in untrusted files should be treated as suspicious, not authoritative.
